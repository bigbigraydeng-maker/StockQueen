术语表

收益指标
策略从起点到终点的总回报率。例如 +85.3% 表示 1 万元变成了 18,530 元。
公式：(期末净值 / 期初净值 - 1) × 100%
将累计收益折算为等效年度收益率，便于不同时间跨度的策略横向对比。
公式：(1 + 累计收益)^(52/周数) - 1
策略收益超过基准（SPY/QQQ）的部分。Alpha > 0 表示策略跑赢了大盘，是主动管理的"附加值"。
公式：策略累计收益 - 基准累计收益
该策略参数组合在当月实际取得的收益率。由当月所有周收益复合计算得出。
Walk-Forward 分析中，被选中参数组合在前 3 个月训练窗口内的累计收益。这是选择该组合的依据——系统挑训练期表现最好的组合用于下一个月。
注意：训练期表现好不保证实际月份也好，这就是市场的不确定性。
风险指标
每承担 1 单位风险（波动率）获得的超额回报。衡量的是"值不值得冒这个险"。
< 1 = 一般  |  1~2 = 好  |  > 2 = 优秀
公式：年化收益 / 年化波动率
从历史最高点到最低点的最大跌幅。代表"最坏情况你会亏多少"。
例如 -25% 表示在最不利时期，10 万元最多缩水到 7.5 万元。
越小越好。一般希望 < 20%。
收益率的标准差，衡量策略收益的"颠簸程度"。波动率越高，单周收益的起伏越大。
公式：周收益标准差 × √52
盈利周数占总周数的比例。50% 以上说明赚钱的周数多于亏钱的周数。
胜率高不一定赚钱（小赚大亏），需配合夏普率综合看。
策略参数
每期持有评分最高的 N 只股票。Top3 = 只买前3名，Top5 = 买前5名。
N 越小越集中（高风险高回报），N 越大越分散（更稳健）。
给已持有的股票额外加分，减少频繁换仓。值越大，换手率越低。
0 = 完全按评分排名换仓（换手率高）
2.5 = 强烈倾向于留住现有持仓（换手率低）
综合 9 个因子的加权总分，范围 [-10, +10]。分数越高越值得买入。
因子权重：动量(25%) > 技术面(15%) = 基本面(15%) > 盈利(10%) = 趋势(10%) = 相对强度(10%) > 现金流(5%) = 情绪(5%) = 板块风向(5%)
市场环境
根据 SPY 的均线和波动率自动判定的市场状态：
STRONG_BULL — 强牛（SPY 在所有均线上方且动量强）
BULL — 牛市（SPY 在多数均线上方）
CHOPPY — 震荡市（方向不明）
BEAR — 熊市（SPY 在均线下方），自动切换到防守持仓（SH、GLD、SHY 等）
基准指数
追踪标普500指数的ETF，代表美国大盘股整体表现。是最常用的投资基准——超过 SPY 才算"有本事"。
追踪纳斯达克100指数的ETF，以科技股为主（苹果、微软、英伟达等）。由于我们的股票池包含大量科技股，QQQ 是更贴切的对比基准。
分析方法
一种避免"事后诸葛亮"的策略优化方法。用过去 3 个月的数据选出最优参数，然后用这个参数在下一个月实际交易。每月滚动更新。
比"全局最优"更真实，因为你不可能提前知道未来。
从 1.0 开始的净值走势图。1.50 = 赚了 50%，0.80 = 亏了 20%。线越陡越赚钱，弯越深越痛苦。
个股表现与 SPY 的比值。RS > 1 表示跑赢大盘，RS < 1 表示跑输。系统过滤掉 RS 过低的标的，只买"比大盘强"的股票。
基于平均真实波幅（ATR）设置的动态止损线。当价格跌破止损价时自动卖出，限制单次亏损。
ATR 越大说明股票波动越大，止损线就设得越宽。
