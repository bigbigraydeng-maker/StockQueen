# StockQueen Images

## Required Images for SEO

### OG Images (Open Graph)
- `og-image.jpg` - 1200x630px, English version for social sharing
- `og-image-zh.jpg` - 1200x630px, Chinese version for social sharing
- Design: Dark background (#0b0f19), StockQueen logo, key stats (94.5% returns)

### Twitter Cards
- `twitter-card.jpg` - 1200x600px, can reuse og-image

### Favicon
- `favicon.ico` - 32x32px or multi-size ICO
- `apple-touch-icon.png` - 180x180px for iOS

### Logo
- `logo.png` - For structured data, ~200x200px transparent PNG

## Design Guidelines

### Color Scheme
- Background: #0b0f19 (dark navy)
- Primary: #6366f1 (indigo) to #06b6d4 (cyan) gradient
- Text: White

### Content for OG Image
```
StockQueen
AI-Powered Quantitative Trading
94.5% Returns Since 2023
```

### Content for Chinese OG Image
```
StockQueen 睿达资本
美股AI量化交易策略
2023年以来收益 94.5%
```

## Tools to Create
- Canva (recommended)
- Figma
- Adobe Photoshop

## Notes
- Keep file size under 500KB for fast loading
- Use WebP format if supported by platform
- Test with Facebook Debugger and Twitter Card Validator
