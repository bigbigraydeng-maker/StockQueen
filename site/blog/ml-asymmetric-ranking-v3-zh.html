ML-V3A：非对称标签如何让2024年收益翻倍、回撤砍半

2026年3月20日 • 约12分钟阅读 • StockQueen研究团队

ML-V2 令人失望。它在崩盘环境中改善了 Sharpe +0.14，但在牛市中系统性拖累收益 -7%——恰恰在进攻型策略最应该发力的时候。 诊断出根本原因后，我们做了一个精准修改。ML-V3A 在2024年实现 +104.4% 收益（对比基准 +46.4%）， 最大回撤从 40.3% 压缩至 8.1%。 以下是我们具体改了什么、为什么有效。

ML层的作用

StockQueen的核心引擎用22维因子向量对500只股票池中的每一只股票评分——动量窗口、成交量趋势、相对强弱、基本面、市场结构信号。得分最高的 Top-N 只股票每周持仓。

ML层叠加在这之上。因子评分计算完成后，XGBoost排名模型对候选前10名重新排序，选出最终6个仓位。这是一个重排过滤器，而非替代因子评分。 核心问题是：模型能否真正学到"前10个因子推荐里，哪只下周会跑赢"？

ML流水线工作方式

1. 因子引擎对500只股票评分 → 选出候选前10名
2. XGBoost排名器接收前10名的22维特征向量
3. 模型输出每个候选的重排分数
4. 最终组合 = ML分数前6名（非原始因子分数前6名）
5. 模型每月1日在18个月滚动数据上自动重训

ML-V2 为什么在牛市里失败

ML-V2 使用对称截面z-score作为训练标签。对于训练数据中的每只股票，我们计算其当周收益，对全宇宙标准化为z-score，然后喂给XGBoost排名器。

问题在于：22个特征权重大致均等（各约0.050），模型对"避免输家"信号和"押注赢家"信号给予同等权重。 在牛市中，上行是非对称的，真正的超额收益来自最大化押注动量领跑者， 而这种对称处理方式系统性地稀释了组合中最优质的选股。

ML-V2根本原因：对称标签问题

V2 对称标签

z_score = (收益 - 均值) / 标准差

+2.0 z-score的赢家和 -2.0 z-score的输家在损失函数中受到等量反向权重。 模型同等地学习"避免输家"和"押注赢家"——但在牛市里，避免输家几乎无关紧要， 重要的是对最优选股有足够的信念。

V3A 非对称标签

z = z_raw × 1.5  当 z_raw > 0
z = z_raw × 0.5  当 z_raw < 0

正向信号放大1.5倍，负向信号压缩为0.5倍。 模型被迫学习"什么让赢家更出色"， 而非在赢家和输家之间平均分配注意力。

非对称标签是刻意为之的归纳偏置。我们在告诉模型： "错过一个输家的代价是选对赢家的一半。" 这将学到的权重向动量放大特征倾斜，恰好是进攻型轮动策略所需要的。

W5 2024：最关键的验证

窗口5（2024 AI牛市）是最重要的验证点。基准有40.3%最大回撤，Sharpe仅0.50——典型的"赚了钱却没睡好觉"。 V3A将回撤压缩至8.1%，同时将收益提升至104.4%。

这不是巧合。在趋势性AI牛市中，非对称标签推动模型集中持有已展现出最强动量信号的股票，同时弱化负向标签降低了模型过早轮出当前赢家的动机。

诚实说明：W2 2021的落后

V3A在W2（2021年牛市巅峰）落后于基准：+174.8% vs +235.0%。这值得解释，而不是回避。

2021年基准取得+235%，是因为我们包含500只以上动量筛选标的的动态股票池自然捕捉了meme股和投机成长股的狂飙。 在那种抛物线式行情中，任何高集中度的激进动量策略都会有异常表现。 ML模型的选择性在极端行情中可能牺牲部分原始收益，换取更好的风险调整表现。

我们认为这是可以接受的：V3A在那个窗口仍然取得+174.8%，Sharpe从3.48提升到3.61。 策略无需抓住每一次投机飙涨，就能创造持续的超额收益。

月度自动重训机制

基于2020–2024数据训练的静态ML模型会随市场环境演变逐渐失效。 我们通过每月1日自动运行的18个月滚动重训窗口解决这一问题。

重训流水线

1. 拉取18个月周度OHLCV数据（AV API，约7分钟）
2. 运行轮动回测，收集每周评分快照
3. 生成非对称z-score标签训练数据
4. 重训XGBRanker → 保存模型 → 内存热更新
5. 飞书通知（特征重要性、排名相关性、Top3特征）

为什么选18个月？覆盖至少一个完整市场Regime周期，同时AV API调用成本可控。全历史重训需30分钟以上，18个月约7分钟。

核心结论

  • ✓ 非对称标签是简单而精准的修复。一行代码（正向×1.5，负向×0.5）带来了V2 Sharpe提升的6倍。
  • ✓ 动态股票池与模型同等重要。V3A在500只动态股票池上持续优于V2在200只静态池上的表现——更多训练数据直接带来更好的排名器校准。
  • ✓ ML是重排增强层，不是独立信号。它有效的前提是底层因子评分质量足够高。因子评分差，再好的ML标签设计也无济于事。
  • ⚠ 2024年结果是强有力的证据，但不保证重现。所有结果来自受控A/B Walk-Forward测试的样本外窗口。历史业绩不代表未来收益。
