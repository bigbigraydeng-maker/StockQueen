- V4 每次独立拉取缓存，MR/ED 运行后往缓存注入新股票 → V4 每次用的宇宙不同
- equity_curve 长度不对齐 → `min_len` 被最短者截断（338天误判为周频）

**修复：**
- `portfolio_manager.py`：预取 OHLCV 一次，V4/MR/ED 三策略共享 `shared_histories`
- `event_driven_service.py`：新增 `fetch_ed_fundamentals_only()`，避免重复拉价格
- `_compute_portfolio_stats`：改为显式 `frequency` 参数
- 修复验证：V4 全期结果 334.7%（修复前：1518% vs 2673%）
- commit: `5ab0ccf`
