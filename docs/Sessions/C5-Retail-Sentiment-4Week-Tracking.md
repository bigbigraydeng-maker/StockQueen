## 一、系统架构

### 多源融合评分

| 数据源 | 权重 | 来源 | 可用性 | 备注 |
|--------|------|------|--------|------|
| Fear & Greed Index | 40% | alternative.me | ✓ | 美国散户情绪，无认证 |
| VIX 恐慌指数 | 35% | CNBC + Yahoo Finance | ✓ | 市场隐含波动率，3层降级 |
| StockTwits 情绪 | 15% | FNG数学推导 | ✓ | 社交平台实时情绪，降级方案 |
| WSB 异常股 | 10% | Reddit WSB 热议 | ✓ | 散户FOMO信号，高噪声 |

### 综合评级逻辑

四源加权投票（百分比制）：

- 极度贪婪 (>=3票) → "extreme_greed" 上限ED配置
- 贪婪上升 (>=2票) → "elevated" 提升ED配置
- 平常状态 (0-1票) → "normal" 标准ED配置
- 恐慌下跌 (<0) → "fear" 降低ED配置

### 系统权重占比

```
破浪 V5 总权重 = 100%
├─ 宝典 V4 趋势轮动: 60-70% (regime动态调整)
├─ ED 事件驱动: 20-30%
├─ MR 均值回归: 0-10%
└─ 现金缓冲: 0-50%

C5 Regime Gate 作用范围：
├─ ED 基础分配: 20-30%
├─ C5 动态调整: ±4-9% (meme_mode 开启时)
└─ 净影响: 11-39% 的 ED 成交量变动
```

**C5 间接权重**: 4-9%（通过gate ED分配）
