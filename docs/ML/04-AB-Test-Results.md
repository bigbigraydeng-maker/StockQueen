## 关键洞察

### W5（2024）是最重要的现实检验 💎

- Baseline 有 **40.3% 回撤**，V3A 压到 **8.1%**（减少 32pp）
- 收益从 46.4% 翻到 **104.4%**，Sharpe 从 0.5 飙到 **3.07**
- 这是"又赚更多、又少亏"的完美组合

### 非对称标签效果已生效

训练数据 top_q_zscore（正标签上限）:
- V2：1.14（对称）
- V3A：1.74（非对称放大上行）✅

bottom_q_zscore（负标签下限）:
- V2：-1.18（对称）
- V3A：-0.55（非对称压缩下行）✅

### W2（2021）落后的合理解释

Baseline 当年 +235%（动态池在极端牛市中表现惊人），ML 仍取得 +174.8%。
在如此高基数下落后 -60pp 是预期内的——这种极端牛市 baseline 难以复现。

## 月度自动重训（2026-03-20 上线）

### 机制

```
每月1日 13:00 NZT
  ↓
run_ml_retrain(months_lookback=18)
  ↓
_fetch_backtest_data(train_start, train_end)     ← AV API，约7分钟
  ↓
run_rotation_backtest(..., collect_snapshots)     ← 收集每周评分快照
  ↓
build_training_data(asymmetric=True)             ← 非对称 z-score 标签
  ↓
MLRanker.train() → save() → 热更新内存实例
  ↓
Feishu 通知（corr / rank_spread / Top3特征）
```

### 关键参数

| 参数 | 值 | 说明 |
|------|----|------|
| `months_lookback` | 18 | 滑动窗口长度（可调，6-36个月）|
| `asymmetric` | True | 与 V3A 完全一致 |
| 触发时间 | 每月1日 13:00 NZT | 在 Auto Param Tune 之后1小时 |
| 手动触发 | `POST /rotation/ml/retrain` | 需 API Key，后台执行 |

### 为什么滑动18个月而不是全历史重训？

1. **计算成本可控**：18个月 ~500只股票 = ~7分钟，全历史需要30分钟+
2. **适应市场环境**：市场 regime 每18个月平均切换1-2次，窗口覆盖至少一个完整周期
3. **无历史包袱**：不让2018-2019的旧数据稀释最新市场信号
4. **Render 无磁盘缓存**：只需拉近期数据，AV API 调用次数可接受
