---

## 因子总览

所有因子评分范围: **[-1, +1]**，最终加权总分范围: **[-10, +10]**

| # | 因子名称 | 默认权重 | 大盘股权重 | 数据来源 | IC验证(3.21) |
|---|---------|---------|-----------|---------|-------------|
| 1 | Momentum (动量) | 0.20 | 0.15 | OHLCV | IC弱有效，稳定 |
| 2 | Technical (技术) | 0.05 | 0.08 | OHLCV | IC全负，有害↓↓ |
| 3 | Trend (趋势) | 0.10 | 0.10 | OHLCV | IC弱正 |
| 4 | Relative Strength (相对强度) | 0.08 | 0.20 | OHLCV + SPY | IC不显著↓ |
| 5 | Fundamental (基本面) | 0.12 | 0.22 | Massive 公司概况/比率 | 无法IC验证↓ |
| 6 | Earnings (盈利) | 0.22 | 0.10 | Massive 财报历史 | t=5.21最强↑↑ |
| 7 | Cashflow (现金流) | 0.13 | 0.05 | Massive 现金流表 | IC=+0.062最高↑↑ |
| 8 | Sentiment (情绪) | 0.05 | 0.05 | Massive 新闻 | 维持 |
| 9 | Sector Wind (板块顺风) | 0.05 | 0.05 | ETF历史 | 维持 |

> **IC验证后调整(2026-03-21)**: earnings+cashflow 合计从15%→35%（IC最强），technical 从15%→5%（IC全负有害）。
> **大盘股独立权重**: 基本面(0.22)和相对强度(0.20)占核心，动量(0.15)和技术(0.08)权重降低。理由: 大盘股基本面稳定，短期动量噪音大。

## 因子 1: Momentum (动量) — 权重 0.20

### 计算公式
```
ret_1w = (close[-1] / close[-6]) - 1          # 1周收益
ret_1m = (close[-1] / close[-22]) - 1         # 1月收益
ret_3m = (close[-1] / close[-63]) - 1         # 3月收益

raw_momentum = w1w * ret_1w + w1m * ret_1m + w3m * ret_3m
volatility = std(daily_returns[-22:]) * sqrt(252)
score = (raw_momentum - 0.50 * volatility) * 3.0
score = clamp(score, -1, +1)
```

### 体制自适应权重
| 市场状态 | 1W权重 | 1M权重 | 3M权重 | 策略意图 |
|----------|--------|--------|--------|---------|
| strong_bull | 0.15 | 0.35 | 0.50 | 偏重长期趋势 |
| bull | 0.20 | 0.40 | 0.40 | 均衡 |
| choppy | 0.35 | 0.40 | 0.25 | 偏重短期反转 |
| bear | 0.40 | 0.40 | 0.20 | 快速反应 |

### 波动率惩罚
- `VOL_PENALTY = 0.50`: 高波动股票被压低分数
- 目的: 偏好"稳定上涨"而非"暴涨暴跌"

## 因子 2: Technical (技术指标) — 权重 0.05

### 5个子指标

| 子指标 | 看多(+1) | 看空(-1) |
|--------|---------|---------|
| RSI(14) | < 30 (超卖) | > 70 (超买) |
| MACD(12,26,9) | histogram > 0 | histogram < 0 |
| Bollinger(20) | 位置 < 0.2 (接近下轨) | 位置 > 0.8 (接近上轨) |
| OBV趋势 | 上升 | 下降 |
| ADX(14) | > 25 时放大主导方向 | 无趋势不调节 |

### 聚合
```
net_score = sum(sub_scores)  # 范围 [-5, +5]
final = clamp(net_score / 3.0, -1, +1)
```

## 因子 4: Relative Strength (相对强度) — 权重 0.08

### 计算公式
```
rs = ticker_return(21d) - SPY_return(21d)
score = clamp(rs * 10.0, -1, +1)
```

### 额外过滤
- **RS < -2% vs SPY → 直接跳过该股票**（不入选TOP N）
- 大盘股权重提升到 0.20（核心因子）

## 权重自动归一化

当某些因子数据不可用时（如 ETF 没有 Earnings 数据），系统自动：
1. 跳过不可用因子
2. 按可用因子重新分配权重，使总权重和 = 1.0
3. 记录 `unavailable_factors` 用于审计

```
例: 如果 Earnings + Cashflow 不可用 (权重 0.15)
其余 7 个因子权重按比例放大: 每个因子 * (1 / 0.85)
```
