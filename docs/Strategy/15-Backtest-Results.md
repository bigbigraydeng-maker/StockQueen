## 参数稳定性分析

### TOP_N（持仓数）
```
W1=6, W2=5, W3=3, W4=2, W5=4, W6=6
→ ❌ 不稳定：2~6 波动大
→ 锁定值: 6（最近两窗口一致，且W1也是6）
```

### HOLDING_BONUS（持仓惯性）
```
W1=0, W2=1.0, W3=0, W4=0, W5=0, W6=0
→ ✅ 稳定：5/6 窗口选择 0
→ 锁定值: 0.0
```

### 动量权重（1W, 1M, 3M）
```
W1=(0.35,0.40,0.25), W2=(0.35,0.40,0.25), W5=(0.35,0.40,0.25)  → choppy型
W4=(0.15,0.35,0.50), W6=(0.15,0.35,0.50)  → strong_bull型
W3=(0.20,0.40,0.40)  → bull型
→ ⚠️ 中等稳定：Regime-specific 权重概念验证成功
```

### ATR 止损参数
```
ATR_STOP: 1.5 (所有窗口)
TRAILING_MULT: 1.5 (所有窗口)  
TRAILING_ACTIVATE: 0.5 (所有窗口)
→ ✅ 完全稳定：已在 Phase A 前锁定
```

## 过拟合风险评估

| 指标 | 值 | 判定 |
|------|-----|------|
| 平均训练Sharpe | 3.16 | |
| 平均OOS Sharpe | 2.49 | |
| 衰减率 | 0.68 | ⚠️ 中等风险 |
| 训练>>OOS 的窗口 | W4 (5.17→1.18) | 最严重 |
| OOS>训练 的窗口 | W1 (0.95→2.82) | 反向验证 |

### 缓解措施
1. **参数锁定**: ATR止损参数预先锁定，不参与搜索
2. **保守选择**: HOLDING_BONUS=0 虽然在W2表现好，但其他窗口都选0
3. **Regime适配**: 不同市场状态用不同动量权重，符合金融直觉
4. **滑点扣减**: 双边 0.1% 滑点已计入回测
5. **流动性过滤**: 50万日均成交量门槛
