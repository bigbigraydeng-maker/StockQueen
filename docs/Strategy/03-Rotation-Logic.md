# 周轮动流程

---

## 轮动 Pipeline（每周一执行）

```
Step 1: 体制检测
    ↓
Step 2: 对全宇宙评分（保存所有分数用于热力图）
    ↓
Step 3: 体制过滤（bear仅防守, choppy仅蓝筹, bull/strong_bull激进）
    ↓
Step 4: 相对强度过滤（RS < -2% vs SPY 跳过）
    ↓
Step 5: 持仓惯性加分（已持有 +0.0, V4锁定为零）
    ↓
Step 6: 板块集中度限制（同板块最多2只）
    ↓
Step 7: 选出 TOP N（N=3）
    ↓
Step 8: 评分加权分配仓位
    ↓
Step 9: 设置止损/止盈
```

---

## Step 1: 体制检测（Regime Detection）

### 4个信号评分

| 信号 | 看多 | 中性 | 看空 |
|------|------|------|------|
| SPY vs MA50 | 上升>2%: +2, 上升0-2%: +1 | — | 下降0-2%: -1, 下降>2%: -2 |
| SPY vs MA20 | 上升: +1 | — | 下降: -1 |
| 21日波动率 | < 12%: +1 | 12-25%: 0 | > 25%: -1 |
| 1月收益率 | > 3%: +1 | -3%~+3%: 0 | < -3%: -1 |

**总分范围: [-5, +5]**

### 映射规则

| 总分 | 体制 | 意味着 |
|------|------|--------|
| >= 4 | strong_bull | 强势上涨，全力进攻 |
| >= 1 | bull | 温和上涨，正常运作 |
| >= -1 | choppy | 震荡，谨慎操作 |
| < -1 | bear | 下跌趋势，防守为主 |

---

## Step 3: 体制过滤

不同体制使用不同的选股池：

| 体制 | 可选池 |
|------|--------|
| strong_bull | 攻击ETF(14) + 大盘蓝筹(~100) + 动态中盘成长（动态池 1,688 只） |
| bull | 同上 |
| choppy | 防守ETF(3) + 攻击ETF(14) + 大盘蓝筹(~100) + 动态中盘成长（动态池 1,688 只） |
| bear | 防守ETF(3) + 反向ETF(4) |

> bear 模式下只从 7 只 ETF 中选，自动规避个股风险

> V5 纯 Alpha 模式：USE_DYNAMIC_UNIVERSE=True，选股池扩展至动态 1,688 只（按 Regime 分类仍保留 ETF 优先规则）

---

## Step 4: 相对强度过滤

```
如果 ticker_return(21d) - SPY_return(21d) < -2%:
    跳过该股票，不参与 TOP N 排名
```

目的: 即使一只股票在其他因子上得分高，如果它最近跑输大盘太多，也不选入。

---

## Step 5: 持仓惯性（Holding Bonus）

```
如果股票已在持仓中:
    total_score += HOLDING_BONUS  (V4锁定: 0.0)
```

**V4 Walk-Forward 结论: HOLDING_BONUS = 0 最优**
- 5/6 个 OOS 窗口选择 0
- 不给持仓股票加分 = 每周完全按实力排名 = 更灵活换仓

---

## Step 6: 板块集中度限制

```
MAX_SECTOR_CONCENTRATION = 2
按评分从高到低选择:
    如果该板块已有 2 只 → 跳过，选下一个
```

目的: 避免持仓全是科技股或全是金融股。

---

## Step 8: 评分加权仓位分配

```python
# 每只股票权重 = 其评分占比
weight[i] = score[i] / sum(all_selected_scores)

# 例: TOP 6 评分为 [8.2, 7.5, 6.8, 6.3, 5.9, 5.1]
# 权重 = [0.205, 0.188, 0.170, 0.158, 0.148, 0.128]
# 第1名仓位是第6名的 1.6 倍
```

特性:
- **SCORE_WEIGHTED_ALLOC = True**: 高分股获得更多仓位
- 自动归一化: 总权重 = 1.0
- 包含滑点: 新建仓扣 0.1%，平仓扣 0.1%

---

## 回测 vs 实盘差异

| 环节 | 回测 | 实盘 |
|------|------|------|
| 入场价 | 次日开盘价 | 次日开盘价（手动/自动） |
| Fundamental 因子 | 禁用（防前视偏差） | 启用 |
| Sentiment 因子 | 禁用（无历史数据） | 启用 |
| 滑点 | 0.1% 固定 | 实际成交价差 |
| 止损执行 | 按止损价精确成交 | 可能有跳空缺口 |
