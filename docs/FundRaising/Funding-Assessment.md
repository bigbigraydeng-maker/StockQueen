## 一、总体结论：融资是否可行？

**四家一致结论：可以融资，但处于 Pre-seed / Angel 阶段，不是机构级项目。**

| 维度 | 评价 |

## 三、项目亮点（四家共识）

### 1. 技术壁垒极高
- Walk-Forward + OOS 验证是机构级方法论
- OOS Sharpe 3.10，5个窗口全部盈利
- 参数锁定 + 展示失败案例（反过拟合），诚实度加分
- OOS/IS > 1，证明非过拟合

### 2. 已建成完整系统（不是 idea）
- Supabase + Render + Tiger API 全栈已跑通
- 21个自动化任务实际运行
- ML + 因子 + Regime 三层架构

### 3. 叙事有吸引力
- "90%基金跑输指数"（SPIVA 数据支撑）
- 2022熊市回撤 -5.57% vs SPY -19%
- "反AI神话"定位（AI不做决策，做辅助）

### 4. 法律文件准备充分（DeepSeek/Gemini 特别提到）
- SAFE、创始人协议、IP转让契约齐全
- 降低投资人尽职调查成本

## 四、项目缺点/风险（四家共识）

### 1. 无实盘 Track Record（致命级）
- 所有数据基于 OOS 回测 + Paper Trading
- 滑点、流动性、执行偏差、心理因素均未验证
- **四家一致认为这是当前最大融资障碍**

### 2. Sharpe 太高反而引发怀疑
- 3.10 远超顶级基金（通常 1.5–2.0）
- 投资人第一反应："要么天才，要么有问题"

### 3. 商业模式零验证
- 无付费用户、无 CAC/LTV/留存数据
- "卖信号"和"管钱"是两门不同的生意

### 4. 合规风险大
- 不持有 AFSL（澳洲）/ FMA（新西兰）牌照
- Signal → Copy Trading → Fund 每一步都是监管雷区
- 牌照成本和时间线不明确

### 5. 团队薄弱
- 仅 2 人（CEO + CMO）
- 缺风控、合规、执行层面的专职人员
- 缺乏大型金融机构 Portfolio Manager 背书

### 6. 公司尚未注册（DeepSeek 指出）
- ACN 未分配，投资人无法打款
- 创始人股权比例尚未确定（协议中为"【●】"）

## 七、融资前必须解决的 3 件事（四家共识）

### 1. 建立 Live Track Record（最优先）
- 自有资金实盘 3–6 个月
- 每周透明披露净值曲线

### 2. 获取第一批付费用户
- 哪怕只有几十个，证明有人愿意买单
- 跑通 CAC/LTV 数据

### 3. 明确合规路径
- Signal vs Financial Advice 界限
- Copy Trading 牌照方案
- Fund 结构（LP/GP）时间表

## 八、各家独特观点

### DeepSeek 独有
- **创始人股权建议**：技术方（邓）贡献核心 IP，应 60/40 或 70/30
- **ESOP 池**：融资前预留 15%–20%
- **IP 转让**：Schedule 2（胡总 IP）需明确列出所有营销/品牌资产

### Gemini 独有
- **对标券商战略投资**：Tiger/富途/IB 可能对"自带策略和流量的技术团队"感兴趣
- **$50 目标 CAC**：需小范围测试验证，投资人会重点追问 LTV/CAC 是否达 5x

### ChatGPT/Grok 独有
- **叙事定位**：你卖的不是策略，是"未来可扩展的量化资产管理平台"
- **实盘后估值翻倍**：$5M → $15M+
