# StockQueen — AI 驱动的美股量化轮动系统

> 用算法代替情绪，让数据驱动每一次决策。

---

## 一句话介绍

StockQueen 是一套 **全自动美股量化投资系统**，通过 AI 多因子评分引擎，每周从 80+ 只精选标的中筛选最强势股票构建投资组合，并根据市场环境智能切换攻守策略。

---

## 核心能力

### 🧠 9 因子 AI 评分引擎
每只股票都经过 9 个维度的量化打分：

| 因子 | 权重 | 数据来源 |
|------|------|----------|
| 动量强度 | 25% | 1周/1月/3月收益率 |
| 技术面 | 15% | RSI/MACD/均线系统 |
| 基本面 | 15% | 营收增长/利润率/PE |
| 盈利质量 | 10% | EPS超预期/连续增长 |
| 趋势确认 | 10% | 均线多头排列 |
| 相对强度 | 10% | 跑赢大盘能力 |
| 现金流 | 5% | 自由现金流/负债率 |
| 市场情绪 | 5% | AI情绪分析 |
| 板块风向 | 5% | 板块轮动信号 |

### 🔄 智能市场环境适应
系统实时识别 4 种市场状态，自动切换策略：

- **🟢 STRONG_BULL（强牛）** → 全仓进攻，买入评分最高的成长股
- **🟢 BULL（牛市）** → 进攻为主，适度分散
- **🟡 CHOPPY（震荡）** → 攻守兼备，配置防御型 ETF
- **🔴 BEAR（熊市）** → 全面防守，自动切换到黄金/国债/做空 ETF

|------|---------|

### 🤖 Walk-Forward 自适应优化
不是简单地回测"什么参数过去最好"（马后炮），而是用 **滚动训练窗口** 模拟真实的决策过程——每月只用前3个月的数据选择下一月的最优参数。

## 技术架构

```
┌─────────────────────────────────────────────┐
│                 StockQueen                    │
│                                              │
│  ┌──────────┐  ┌──────────┐  ┌───────────┐  │

│  └──────────┘  └──────────┘  └───────────┘  │
│       ↑                           ↓          │
```

- **后端**：Python + FastAPI + APScheduler（20个定时任务）
- **前端**：TailwindCSS + HTMX（零JS框架，极速响应）
- **数据**：Alpha Vantage Premium API（实时+历史）
- **部署**：Render 云托管，7×24 运行
